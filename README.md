Conflicts often occur because of leftover files from previous installation attempts or strict security settings in Windows. Follow these steps to prepare your system.

## 🔍 Fixing Common Errors

If the installer stops or shows an error code, use these solutions to move forward.

### Installation Failed: Setup Error
This usually indicates a missing library file. Windows 11 requires updated C++ Redistributables. Visit the Microsoft support website to download the latest Visual C++ Redistributable for Visual Studio. Install both x86 and x64 versions. Restart your computer after you install these support files and try the Sound Forge installer again.

### Insufficient Permissions
If the installer claims it lacks permission, navigate to the folder where your installer file sits. Right-click the folder, go to Properties, then the Security tab. Ensure your user account has "Full Control" or "Modify" permissions checked. Click Apply and try the setup again.

### Compatibility Mode
If the installer exits without a message, right-click the setup file and choose Properties. Select the Compatibility tab. Check the box labeled "Run this program in compatibility mode for" and choose "Windows 10" from the dropdown menu. Click Apply and run the installer again.

### Sound Driver Conflicts
Audio software requires exclusive access to sound hardware. Close any other application using your sound card, such as web browsers, music players, or video conferencing tools, before you launch Sound Forge for the first time.
